# Stats

A simple command-line tool that calculates the number of files and total size in a directory.

---

- Count files.
- Calculate total size.
- Show the largest N files.

## Installation

- Install the package from the project root.

```bash
pip install .
```

## Usage

- Count files, calculate total size and show the largest 3 files in a path.

```bash
stats /path --top 3
```
